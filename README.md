# 📋 App de Organização Pessoal

> Ferramenta web para gerenciar tarefas e rotinas do dia a dia de forma simples e eficiente.

## 📋 Descrição

Aplicação criada para centralizar a organização pessoal — desde tarefas do dia a metas de médio prazo — tudo em uma interface limpa e fácil de usar no cotidiano.

## ⚙️ O que o app faz

- Gerencia tarefas e rotinas do dia a dia
- Permite criar, editar e concluir tarefas
- Organiza atividades por prioridade
- Mantém o histórico do que foi feito
